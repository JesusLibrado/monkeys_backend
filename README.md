# Monkeys backend

## Description

 - [Nest](https://github.com/nestjs/nest)
 - [Prisma](https://www.prisma.io/docs/orm/)
 - [GraphQL](https://docs.nestjs.com/graphql/quick-start#schema-first)

## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# local
$ yarn run start:local
```

#### Using shell script
```bash
# local
$ ./start_local.sh
```
